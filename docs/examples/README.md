## Recommended Learning Path

### New to CoordMCP?

1. Start with **[Basic Project Setup](basic-project-setup.md)**
   - Learn fundamental concepts
   - Create your first project
   - Record architectural decisions

2. Move to **[Architecture Recommendation](architecture-recommendation.md)**
   - Get design guidance
   - Learn about patterns
   - Validate your code

3. Try **[Context Switching](context-switching.md)**
   - Manage multiple projects
   - Switch between tasks
   - Track your work history

4. Master **[Multi-Agent Workflow](multi-agent-workflow.md)**
   - Coordinate with other agents
   - Prevent file conflicts
   - Work as a team

## Example Format

Each example includes:

- **Introduction** - What you'll learn
- **Prerequisites** - What you need to know
- **Scenario** - Real-world use case
- **Step-by-step guide** - Detailed walkthrough
- **Complete code** - Full runnable example
- **Expected output** - What you should see
- **Key concepts** - Important takeaways
- **Troubleshooting** - Common issues

## Code Style

All examples use:
- **Python async/await** syntax
- **Inline code** you can copy and run
- **Real-world scenarios** you can relate to
- **Progressive complexity** building on previous examples
